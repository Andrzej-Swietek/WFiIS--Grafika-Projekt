Detailed Description

Class representing a curve shape derived from Shape.

This class inherits from Shape and represents a curve defined by its stroke width, outline style and color, fill color, and a vector of points that define the curve's path. It provides methods to draw the curve on a device context, rotate it, retrieve its center, manipulate its points, and convert it to a string representation.
