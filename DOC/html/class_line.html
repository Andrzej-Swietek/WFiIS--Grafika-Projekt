Detailed Description

Class representing a line shape derived from Shape.

This class inherits from Shape and represents a line defined by its starting and ending points, stroke width, outline style and color, and fill color. It provides methods to draw the line on a device context, rotate it, retrieve its center, manipulate its start and end points, and convert it to a string representation.
